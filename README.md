# Parley

Landing page premium.

## Estructura

```
parley/
├── index.html              # Entrada — redirige a screens/home
├── README.md
├── assets/
│   └── logo/
│       └── logo.svg
└── screens/
    └── home/
        ├── index.html
        ├── index.css
        └── index.js
```

## Desarrollo

Abrir `index.html` en el navegador o usar un servidor local:

```bash
npx serve .
```
